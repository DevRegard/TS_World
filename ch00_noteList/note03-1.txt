* let, const
 - let: 변할 수 있는 변수
 - const: 변할 수 없는 변수

* 타입 주석
 - let 변수 이름: 타입 [= 초깃값]
 - const 변수 이름: 타입 = 초깃값

* 타입 추론
 - 목적: JS 호환을 위해 타입 부분 생략
 - 이후에 각 변수는 해당 타입만 저장 가능

* any 타입
 - 어떤 종류의 값도 저장 가능

* undefined 타입
 - 타입스크립트에서는 '타입이기도 하고 값' (JS는 값일뿐)
 - any는 최상위 타입, undefined는 최하위 타입

* 템플릿 문자열
 - 목적: 변수에 담긴 값을 조합하여 문자열을 만들기 위해
 - 선언: `${변수 이름}`
 - 문자열에서 문자열로 치환
