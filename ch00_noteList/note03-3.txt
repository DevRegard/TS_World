* 클래스 선언
 - 다른 언어와 문법적인 차이만 약간 있음

* 접근 제한자
 - public, private, protect
 - 만약 생략하면 모두 public

* 생성자
 - 타입스크립트만의 특별한 문법: 타 언어 비해 가장 쉬운 선언
 - ex) construct(public name: string, public age?: number) {}

* 인터페이스 구현
 - 특이사항: 타입을 인터페이스로 선언 가능

* 추상 클래스
 - 특이사항: 추상클래스에서 생성자 선언 가능
 - 추상 클래스는 new 연산자로 호출 불가

* 클래스의 상속
 - 특이사항: 부모 클래스의 생성자를 'super' 키워드로 호출 가능

* static 속성
 - 정적인 속성
 - 구현: static 정적 속성 이름: 속성 타입 [= 초기화]
 - 사용: 클래스 이름.정적 속성 이름
